>**Theorem 3.5.** Let $K$ be a connected simplicial complex. Then
>$$H_0 (K)\cong\mathbb{Z}$$
>
>**Proof**
>$$\begin{align}[z]=\left[\sum_i n_i p_i\right]&=\sum_i n_i [p_i]=\sum_i n_i [p_1]\\&=\left[\left(\sum_i n_i\right)p_1\right] \end{align}$$
